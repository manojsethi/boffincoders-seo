# 06. Phase 3: Page Content Analysis

## Phase Goal

Create a page-level workspace that tells an analyst how to improve a page for its target keywords and business goal.

## Page Content Analysis Should Answer

For each important page:

1. What is this page trying to do?
2. Which keywords does it support?
3. Does content match intent?
4. Is the page strong enough?
5. What sections are missing?
6. Is the CTA appropriate?
7. Is proof/trust strong enough?
8. What internal links should be added?
9. What schema is needed?
10. Should the page be updated, rewritten, merged, or left alone?

## Inputs

- page markdown.
- title/meta/H1/headings.
- page role.
- mapped keywords.
- GSC rows.
- GA4 metrics.
- CWV metrics.
- audit issues.
- internal links in/out.
- schema status.
- project goals.

## Page Analysis Sections

### 1. Page Purpose

Show:

- inferred purpose.
- target audience.
- CTA.
- confidence.

### 2. Keyword Fit

Show:

- mapped keywords.
- intent.
- ranking URL.
- fit verdict.
- evidence.

### 3. Missing Sections

Examples:

- pricing/plan info.
- deliverables.
- comparison.
- FAQ.
- proof/case study.
- eligibility/admission.
- donation impact.
- author/reviewer.
- location details.

### 4. Trust And Proof

Check for:

- testimonials.
- case studies.
- author/reviewer.
- credentials.
- citations.
- process.
- policies.
- organization information.

### 5. CTA And Conversion Fit

Check:

- CTA clarity.
- CTA placement.
- form/contact path.
- mismatch between intent and CTA.

### 6. Internal Link Recommendations

Suggest:

- pages that should link to this page.
- pages this page should link to.
- anchor text candidates.

### 7. Recommended Fix Plan

Generate:

- top 3-10 changes.
- owner.
- impact.
- effort.
- validation.

## Verdicts

Page-level verdict:

- `healthy`.
- `minor_update`.
- `must_improve`.
- `wrong_target`.
- `needs_rewrite`.
- `merge_or_prune`.
- `monitor`.

## AI Use

Local AI:

- summarize page.
- identify missing sections.
- classify intent.
- suggest internal link topics.

External AI:

- rewrite sections.
- generate new content draft.
- produce final content brief.

## Validation Checklist

Do not mark complete until:

- page analysis works for service, article, NGO/program, education/course, and legal pages.
- must-improve verdicts are evidence-backed.
- analyst can create recommendations from page analysis.
- content suggestions do not invent facts.
- self-audit is written.
